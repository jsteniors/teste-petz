# Teste Petz
Aplicação para CRUD de PET e Cliente

## Getting started
### Requirements
* maven 3+
* docker ou java 8+

## Running

Para executar o projeto basta execuar os seguintes comandos

```shell
mvn spring-boot:run
```
ou

```shell
docker run -p 8080:8080 jstenio/teste-petz
```

## Features

* Edição
* Cadastro
* Exclusão
* Visualização

Para as entidades

* Cliente
* Pet
